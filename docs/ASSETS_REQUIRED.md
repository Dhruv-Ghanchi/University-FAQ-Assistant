# Required Assets Checklist

The user must provide the following screenshots in this folder (`docs/assets/`).
These are referenced in the README and documentation.

| Screenshot | Filename | Description |
|------------|----------|-------------|
| **Home UI** | `ui-home.png` | The main chat interface when the app loads. |
| **Disambiguation Options** | `ui-clarification.png` | The chat showing clarification options (e.g., "Do you mean Attendance or Scholarship?"). |
| **Eligibility Modal** | `ui-eligibility-modal.png` | The popup modal form for checking eligibility. |
| **Eligibility Result** | `ui-eligibility-result.png` | The result badges (Eligible/Not Eligible) shown after simulation. |
| **Metrics Summary** | `metrics-summary.png` | A screenshot of the `/metrics/summary` JSON output or a dashboard view if available. |
| **Architecture Diagram** | `architecture-diagram.png` | (Optional) A diagram showing the request flow. |

## Instructions
1. Take screenshots of your running application.
2. Rename them exactly as listed above.
3. Place them in this folder (`docs/assets/`).
4. They will automatically appear in the README.
